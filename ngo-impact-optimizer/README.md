# NGO Impact Optimizer

A full-stack data science platform to analyze and optimize the impact of non-governmental organizations.

## Project Structure

- `backend/`: Node.js/Express API.
- `frontend/`: React-based user interface.
- `data-science/`: Python scripts for data analysis and modeling.
- `dataset/`: Contains NGO impact data in CSV format.

## Getting Started

### Backend
1. `cd backend`
2. `npm install`
3. `npm start`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm start`

### Data Science
1. `cd data-science`
2. `pip install -r requirements.txt`
3. `python analysis.py`
